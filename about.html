About This Site

June 17, 2014

This site is a place for me to publish my thoughts and web experiments. I've written all the code here ( HTML, CSS, RSS ) by hand both as a mental exercise as well as to avoid relying on any server-side code. No blogging engine, no static site generator, no scripts.
